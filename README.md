# Classicthesis Typst Ja

Classic Thesis style template customized for Japanese theses, built for Typst.


## Overview

This is a Typst template based on the renowned [Classic Thesis](https://bitbucket.org/amiede/classicthesis/) style, heavily customized for Japanese typesetting. It is designed to work seamlessly with the [Typst Web App](https://typst.app/) as well as local CLI environments.


## Usage


1.  Upload the entire project folder to [Typst Web App](https://typst.app/).
2.  Open `main.typ` to start writing.
3.  Configure your project details (Title, Author, etc.) in `config.typ`.
